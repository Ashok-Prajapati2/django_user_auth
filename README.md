## Installation

### Prerequisites

- Python
- Django

### Clone Repository

```bash
git clone https://github.com/Ashok-Prajapati2/django_user_auth.git

cd django_user_auth
chmod +x requirement.txt
pip install -r requirement.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
